  # File Search Overlay                                                     
                                                                            
  ## Context                                                                
                                                                            
  gracilius TUIにファイル検索機能を追加する。                               
  `/` キーでオーバーレイを表示し、ファイル名を入力して                      
  部分一致で検索する。選択したファイルは新しいタブで開く。                  
  `bubbles/list.Model` を使ってフィルタリング・カーソル・                   
  スクロールをライブラリに委譲する。

  ## Step 1: `internal/tui/search.go` 新規作成                              
                                                                            
  ### fileItem 型                                                           
                                                                            
  `list.Item` と `list.DefaultItem` を満たす型。                            
                                                                            
  ```go                                                                     
  type fileItem struct {                                                    
      path string // rootDir相対パス                                        
  }                                                                         
                                                                            
  func (f fileItem) Title() string       { return f.path }                  
  func (f fileItem) Description() string { return "" }                      
  func (f fileItem) FilterValue() string { return f.path }                  
  ```                                                                       
                                                                            
  ### searchOverlay 構造体                                                  
                                                                            
  ```go                                                                     
  type searchOverlay struct {                                               
      active bool                                                           
      list   list.Model                                                     
  }                                                                         
  ```                                                                       
                                                                            
  ### 主要関数                                                              
                                                                            
  - `newSearchOverlay() searchOverlay`                                      
    - 空の `list.Model` を初期化                                            
    - 不要UIをオフ:                                                         
      `SetShowTitle(false)`,                                                
      `SetShowStatusBar(false)`,                                            
      `SetShowHelp(false)`                                                  
    - delegate: `list.NewDefaultDelegate()` に                              
      `ShowDescription = false`,                                            
      `SetHeight(1)` を設定                                                 
    - `SetFilteringEnabled(true)`                                           
    - `KeyMap` から不要バインドを無効化                                     
      (Quit, ForceQuit等)                                                   
  - `scanAllFiles(rootDir string) []list.Item`                              
    - `filepath.Walk` で再帰スキャン                                        
    - 既存の `isHiddenEntry()` (filetree.go) を再利用                       
    - ファイルのみ返す(ディレクトリ除外)                                    
    - rootDir相対パスの `fileItem` スライスで返す                           
  - `(s *searchOverlay) open(rootDir string)`                               
    - `scanAllFiles` でアイテムを取得                                       
    - `s.list.SetItems(items)` で設定                                       
    - フィルタモードを開始                                                  
      (`s.list.FilterInput.Focus()` 等)                                     
    - `s.active = true`                                                     
  - `(s *searchOverlay) selectedPath() string`                              
    - `s.list.SelectedItem()` から                                          
      `fileItem.path` を取得                                                
  - `(s *searchOverlay) view(width, height int) string`                     
    - `s.list.SetSize(overlayW, overlayH)` でサイズ設定                     
    - lipglossでボーダー付きボックスを描画                                  
    - `lipgloss.Place(width, height, Center, Center, box)`                  
      で画面中央に配置

  ### tea.KeyMsg にsearchモードブロック追加                                 
                                                                            
  `inputMode` チェックの後、通常キー処理の前に挿入:                         
                                                                            
  ```go                                                                     
  if m.search.active {                                                      
      switch {                                                              
      case key.Matches(msg, m.keys.Cancel):                                 
          m.search.active = false                                           
          return m, nil                                                     
      case msg.Type == tea.KeyEnter:                                        
          if item := m.search.selectedPath(); item != "" {                  
              absPath := filepath.Join(m.rootDir, item)                     
              m.search.active = false                                       
              m.openFileByPath(absPath)                                     
          }                                                                 
          return m, nil                                                     
      default:                                                              
          // list.Model に委譲                                              
          var cmd tea.Cmd                                                   
          m.search.list, cmd = m.search.list.Update(msg)                    
          return m, cmd                                                     
      }                                                                     
  }                                                                         
  ```                                                                       
                                                                            
  ### `/` キーハンドラ追加

  case key.Matches(msg, m.keys.Search):
